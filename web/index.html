The full neural text-to-speech stack — duration, acoustic, and iSTFT decoder — compiled to WebAssembly and run entirely on this page. No server, no network calls after load, no WebGPU. The same C99 core that runs on an ESP32 microcontroller, in ~600 KB of int8 weights.

The speed shown is this browser on your CPU, not a microcontroller measurement — a laptop runs this far faster than real time. The point of the demo is that the identical bit-exact runtime executes client-side; the per-device MCU numbers (ESP32-S3 at 0.22× real time, etc.) live in the repo. The verification score is a live Pearson correlation of this run's PCM against the PyTorch reference audio, computed in your browser.
